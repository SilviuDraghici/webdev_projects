to have the emulator save and load all data:
firebase emulators:start --import .\data\ --export-on-exit

export auth from prod:
firebase auth:export save_file.json --format=json

Import database from prod again:

use
https://console.cloud.google.com/storage/browser
to delete existing export

gcloud firestore export gs://learningfullstack-6df15.appspot.com/database_export

gsutil -m cp -r gs://learningfullstack-6df15.appspot.com/database_export .

firebase emulators:start --import .\database_export\

General instructions:
1. Login to firebase and Gcloud:

firebase login
gcloud auth login


2. See a list of your projects and connect to one:

firebase projects:list
firebase use your-project-name

gcloud projects list
gcloud config set project your-project-name


3. Export your production data to gcloud bucket with chosen name:

gcloud firestore export gs://your-project-name.appspot.com/your-choosen-folder-name


4. Now copy this folder to your local machine, I do that in functions folder directly:

cd functions
gsutil -m cp -r gs://your-project-name.appspot.com/your-choosen-folder-name .


5. Now we just want to import this folder.

firebase emulators:start --import ./your-choosen-folder-name
